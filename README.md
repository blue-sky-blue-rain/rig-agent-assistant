# Rig Agent - 文件操作助手

这是一个基于Rig框架和DeepSeek AI的终端文件操作助手。

## 功能

- 创建文件
- 删除文件
- 读取文件
- 执行系统命令（在前四个工具无法完成任务时使用）

## 特点

- 用户确认机制：每次执行操作前都会请求用户确认
- 跨平台支持：Windows和Unix-like系统（Linux/macOS）均可运行
- 简洁的交互界面：使用彩色终端输出，提升用户体验

## 使用方法

1. 克隆项目并进入目录
2. 复制`.env.example`为`.env`，并填入你的DeepSeek API密钥
3. 运行`cargo run`
4. 根据提示输入命令
